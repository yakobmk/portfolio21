Netflix Clone

For this web app, I used react to develop the front end and used firebase to host the site. Major functionality for this website are usage of API keys to display trailer feature for movies recieved from YouTube's database.

The challenge

The purpose of this project was to develop a Netflix Clone web app that is used to stream videos using a database collection of movies from YouTube using React JS and firebase.

Landing Page

Creating a landing page as Netflix had it meant having to showcase a featured movie or series at the top of the home page. This creates an environment where users are able to learn the platform using this featured product as an example creating a more comfortable environment while using Netflix.

Options & Journey

The second imitation taken from netflix is the options journey based on the movie genre. This is designed through React JS by filtering the incoming data to their appropriate movie type which was then followed by css styling that allowed users to navigate for a specific genre by scrolling vertically, and scrolling horizontally to find films or series within a specific genre.

Asset Creation & Feel

Another imitation that I took is the ability to play trailers. This was done by requesting data from YouTube's database through an API call to collect display movie trailers and data on click.
